So What are the main benefits of Using GitHub?

  1. It makes it easier to contribute to your open source projects

    2. To be honest, nearly every open source project uses GitHub to manage its project. Using GitHub is free if your poject is open source and includes a wiki and issue tracker that makes it easy to include more indepthe documentation and get feedback about your project. If you want to contribute, you just fork a project, make your changes and then send them a pull request using the GitHub web interface.

  2. Documentation

    3. By using GitHub, you make it easier to get excellent documentation. Their help sections and guides have articles for nearly any topic related to git that you can think of.

    4. Are you a developer and wish to attract recruiters? Github is the best tool you can rely on for this. Today, when searching for new recruits for their projects, most companies look into GitHub profiles. If your profile is available, you will have a higher chance of being recruited even if you are not from a great University or College.
